**Task: Frontend - User Authentication Implementation**

**Description:**

Implement user registration and login functionality for the Mutual Fund Tracker application. This will allow users to create secure accounts and access their investment data.

**Requirements:**

*   Implement user registration with email and password.
*   Implement user login with email and password.
*   Secure user accounts with password protection.
*   Use Angular for the frontend implementation.
*   Integrate with Supabase for user authentication.
*   Ensure responsive design for various screen sizes.

**Acceptance Criteria:**

*   Users should be able to register with a valid email address and password.
*   Users should be able to log in with their registered email address and password.
*   Passwords should be securely stored in the database.
*   The user interface should be intuitive and easy to use.
*   The layout should be responsive and adapt to different screen sizes.
